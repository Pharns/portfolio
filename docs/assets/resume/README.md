# Resume Files

Place resume PDF files in this directory.

## Expected files

- `Pharns_Genece_Resume.pdf` — Primary resume (ATS-optimized)
- `Pharns_Genece_Resume_GRC.pdf` — GRC-focused variant (optional)
- `Pharns_Genece_Resume_Detection.pdf` — Detection Engineering variant (optional)

## Naming convention

Use underscores, no spaces: `Pharns_Genece_Resume.pdf`

## Link format

Portfolio pages link to: `assets/resume/Pharns_Genece_Resume.pdf`

## Update checklist

When updating resume:
1. Replace PDF file in this directory
2. Verify file name matches links in `contact.md` and `about.md`
3. Run `mkdocs build --strict` to verify
4. Commit and push
